1️⃣ What is the difference between var, let, and const?
2️⃣ What is the spread operator (...)?
3️⃣ What is the difference between map(), filter(), and forEach()?
4️⃣ What is an arrow function?
5️⃣ What are template literals?


//Answer to the qus no: 1-
var: This is the older way to declare variables in JavaScript. It is function-scoped and the same variable can be redeclared and updated.
let: This is the modern way to declare variables. It is block-scoped, cannot be redeclared in the same scope, but its value can be updated.
const: This is used for variables whose value should not be reassigned. It is also block-scoped and must be initialized at the time of declaration.


//Answer to the qus no: 2-
The spread operator (...) is used to expand or spread elements of an array or object.
It allows you to copy, merge, or pass elements easily.
It is commonly used to combine arrays or clone objects. 


//Answer to the qus no: 3-
map(): Executes a function on each element of an array and returns a new array with the transformed values.
filter(): Executes a condition on each element and returns a new array with elements that pass the condition.
forEach(): Executes a function for each element of an array but does not return a new array.

//Answer to the qus no: 4-
An arrow function is a shorter syntax for writing functions in JavaScript.
It uses the => (arrow) syntax instead of the function keyword.
Arrow functions also do not have their own this context.


//Answer to qus no: 5-
Template literals are strings written using backticks ( ) instead of single or double quotes.
They allow string interpolation using ${} to insert variables.
They also support multi-line strings.
